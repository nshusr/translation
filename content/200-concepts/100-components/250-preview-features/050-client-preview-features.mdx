## 启用 Prisma Client 预览功能

要启用 Prisma Client 预览功能，请执行以下操作：

1. 将预览功能标记添加到 `generator` 块中:

   ```prisma
   generator client {
     provider        = "prisma-client-js"
     previewFeatures = ["orderByRelations"]
   }
   ```

2. 重新生成客户端:

   ```terminal
   npx prisma generate
   ```

3. 如果您使用的是 Visual Studio Code，在生成客户端后，`.ts` 文件中的预览功能是不可用的。这时，请运行 **TypeScript: Restart TS server** 命令。
