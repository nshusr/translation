<TopBlock>

Prisma CLI 提供以下预览功能：

- [`db seed`](../../../reference/api-reference/command-reference#db-seed-preview)

要使用预览 CLI 功能，请在运行命令时[附加上 `--preview-feature `标志](./)。[您可以在 GitHub 上对预览功能进行反馈](https://github.com/prisma/prisma/issues/3108)。

</TopBlock>

## 启用 Prisma CLI 预览功能

要使用 Prisma CLI 预览功能，请在每次运行命令时附加上 `--preview-feature` 标志:

```terminal
npx prisma db seed --preview-feature
```
