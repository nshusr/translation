<TopBlock>

要使用 Docker 运行 Microsoft SQL Server 容器镜像，请执行以下操作:

1. 安装和设置[Docker](https://docs.docker.com/get-docker/)
1. 在终端中运行以下命令下载 Microsoft SQL Server 2019 镜像：

   ```terminal
   docker pull mcr.microsoft.com/mssql/server:2019-latest
   ```

1. 创建容器镜像实例，将 `SA_PASSWORD` 的值替换为您的密码:

   ```terminal
   docker run --name sql_container -e 'ACCEPT_EULA=Y' -e 'SA_PASSWORD=myPassword' -p 1433:1433 -d mcr.microsoft.com/mssql/server:2019-latest
   ```

1. [按照 Microsoft 的说明连接到 SQL Server 并使用 `sqlcmd` 工具](https://docs.microsoft.com/en-us/sql/linux/quickstart-install-connect-docker?view=sql-server-ver15&pivots=cs1-cmd#connect-to-sql-server)，将镜像名称和密码替换为您自己的名称和密码。

1. 在 `sqlcmd` 命令提示符下，创建一个新数据库:

   ```terminal
   CREATE DATABASE quickstart
   GO
   ```

1. 运行以下命令检查数据库是否已成功创建:

   ```terminal
   sp_databases
   GO
   ```

</TopBlock>

## 连接 URL 凭证

根据此示例，您的凭证为:

- **Username**: sa
- **Password**: myPassword
- **Database**: quickstart
- **Port**: 1433
