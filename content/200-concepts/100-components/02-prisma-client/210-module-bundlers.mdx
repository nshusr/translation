## 概述

_模块绑定器_ 将 JavaScript 模块捆绑到单个 JavaScript 文件中。大多数绑定器通过将 JavaScript 代码从各种源文件复制到目标文件中来工作。

由于 Prisma Client 不仅基于 JavaScript 代码，而且还依赖于 [**查询引擎二进制文件**](../prisma-engines/query-engine#the-query-engine-binary-file) 可用，因此你需要确保捆绑代码可以访问二进制文件。

为此，你可以使用允许你复制静态资源的插件：

| 绑定器 | 插件                                                                                                  |
| :------ | :------------------------------------------------------------------------------------------------------ |
| Webpack | [`copy-webpack-plugin`](https://github.com/webpack-contrib/copy-webpack-plugin#copy-webpack-plugin)     |
| Parcel  | [`parcel-plugin-static-files-copy`](https://github.com/elwin013/parcel-plugin-static-files-copy#readme) |
