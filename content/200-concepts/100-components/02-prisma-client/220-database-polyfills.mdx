<TopBlock>

Prisma Client 提供的功能通常无法通过特定数据库实现，或者需要扩展。这些特征称为 _polyfills_ 。对于所有数据库，这包括：

- 使用 `cuid` 和 `uuid` 值初始化 [ID](../prisma-schema/data-model#defining-an-id-field) 值

- 使用 [`@updatedAt`](../prisma-schema/data-model#defining-attributes) 存储记录上次更新的时间

对于关系数据库，这包括：

- [隐式多对多关系](../prisma-schema/relations/many-to-many-relations#implicit-many-to-many-relations)

对于 MongoDB，这包括：

- [一般关系](../prisma-schema/relations/) - 文档之间的外键关系在 MongoDB 中不强制执行
</TopBlock>
