## 需要帮助?

Prisma 是由热情的应用开发人员支持的大型[社区](https://prisma.io/community) ，在 [Slack](https://slack.prisma.io) 和 [GitHub](https://github.com/prisma/prisma)最为活跃。

## 关于 Prisma 的一个问题

你可以在 GitHub 的存储库中提出问题并开始[讨论](https://github.com/prisma/prisma/discussions/) 与`prisma`相关的主题.

<ButtonLink
  color="dark"
  type="primary"
  href="https://github.com/prisma/prisma/discussions"
>
  提个问题
</ButtonLink>
<br />
<br />

## 为 Prisma 创建一个错误报告

如果你发现错误信息或者遇到问题，请务必创建一个错误报告！你可以在文档中找到[错误报告的最佳实践](创建错误报告) (如包括其他调试输出)。

<ButtonLink
  color="dark"
  type="primary"
  href="https://github.com/prisma/prisma/issues/new?assignees=&labels=&template=bug_report.md&title='Create bug report'"
>
  创建错误报告
</ButtonLink>
<br />
<br />

## 提交功能请求

如果当前 Prisma 没有特定功能，请务必查看 [路线图](../about/roadmap) ，看看是否已经规划。

如果路线图上的功能与 GitHub 问题相关联，请务必在这个问题上留下+1，最好评论你对该功能的想法！

<ButtonLink
  color="dark"
  type="primary"
  href="https://github.com/prisma/prisma/issues/new?assignees=&labels=&template=feature_request.md&title='Submit feature request'"
>
  提交功能请求
</ButtonLink>
