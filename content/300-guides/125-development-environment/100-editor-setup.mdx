<TopBlock>

本页描述了在使用 Prisma 时，你如何配置你的编辑器以获得最佳的开发者体验。

如果你在这里没有看到你的编辑器，发起[需求](https://github.com/prisma/prisma/issues/new?assignees=&labels=&template=feature_request.md&title=)，为你的编辑器提供专门的支持（例如，语法高亮和自动格式化）。

</TopBlock>

## VS Code

你可以安装插件 [Prisma VS Code extension](https://marketplace.visualstudio.com/items?itemName=Prisma.prisma).

## Community projects

> **Note**: 社区项目没有得到 Prisma 的维护或官方支持，一些功能可能不同步。请自行决定使用。

### Emacs

你可以使用[emacs-prisma-mode](https://github.com/pimeys/emacs-prisma-mode).

### Vim

[vim-prisma](https://github.com/pantharshit00/vim-prisma) 是一个插件，为 Prisma 2 提供文件检测和语法高亮支持。

### Jetbrains IDE

[Jetbrains](https://plugins.jetbrains.com/plugin/14240-prisma) 提供了一个支持 Prisma Schema 语言的扩展。
