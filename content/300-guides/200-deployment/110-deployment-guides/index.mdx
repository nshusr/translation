## 无服务器和 PaaS 平台

以下是不断发展的无服务器和 PaaS（平台即服务）平台列表，Prisma 可以被部署到这些平台上。

### Serverless Platforms

- [AWS Lambda](deploying-to-aws-lambda)
- [Serverless Framework, AWS Lambda](deploying-to-aws-lambda)
- [Netlify](deploying-to-netlify)
- [Vercel](deploying-to-vercel)
- [Azure functions](deploying-to-azure-functions)
- Firebase functions
- GCP functions

### Paas Platforms

- CodeSandbox
- [Heroku](deploying-to-heroku)
- Render
- Railway
- Fly

### Common combinations

Prisma 也被用于一些主要的框架，如[Redwood](https://redwoodjs.com/)和[Blitz](https://blitzjs.com/)。以下是常见组合的列表。

- Vercel + Next
- Vercel + Redwood
- Netlify + Redwood
- Vercel + Blitz
- Heroku + Blitz
- Render + Blitz
