Prisma Client 依赖于[查询引擎](./././concepts/components/prisma-engines/query-engine)。该引擎作为二进制文件与你的应用程序在同一主机上运行。

该查询引擎是用 Rust 实现的，并以可执行二进制文件的形式被 Prisma 使用。二进制文件在调用`prisma generate`时被下载。

例如，如果你在 Windows 机器上开发了你的应用程序，并希望上传到 AWS Lambda（这是一个 Linux 环境），你可能会遇到问题，并在终端出现一些警告。

为了解决这个问题，如果你提前知道你将部署到不同的环境，你可以使用[二进制目标](/concepts/components/prisma-schema/generators#the-native-binary-target)并指定哪些[支持的操作系统](/reference/api-reference/prisma-schema-reference#binarytargets-options)二进制文件应该被包含。

> **注意**: 如果你的操作系统不被支持，你可以包括一个[自定义二进制](.../.../concepts/components/prisma-engines#using-custom-engine-binaries)。

</TopBlock>
